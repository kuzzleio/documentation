# WebSocket class getters

## getPort

Returns the port number used by the protocol instance.

### Arguments

```cpp
unsigned int getPort()
```

## isSslConnection

Returns a boolean indicating if the protocol instance is using SSL or not.

### Arguments

```cpp
bool isSslConnection()
```
