# stopQueuing

Stops the requests queuing. Works only in offline mode, and if the [autoQueue](/sdk/android/3/core-classes/kuzzle/#properties) option is set to `false`.

---

## Return Value

Returns the `Kuzzle` SDK object to allow chaining.
