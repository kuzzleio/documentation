# whoAmI

Returns informations about the user who is currently loggedin.

---

## whoAmI(callback)

| Arguments  | Type     | Description                    |
| ---------- | -------- | ------------------------------ |
| `callback` | function | Callback handling the response |

---

## Callback Response

Returns an instantiated [User](/sdk/php/3/user) object.
