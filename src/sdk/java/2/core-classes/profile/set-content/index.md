# setContent

Replaces the content of the `Profile` object.

:::info
Updating a profile will have no impact until the `save` method is called
:::

---

## setContent(data)

| Arguments | Type        | Description     |
| --------- | ----------- | --------------- |
| `data`    | JSON Object | Profile content |

---

## Return Value

Returns the `Profile` object.
