# startQueuing

Starts the requests queuing.  
Works only in `offline` state, and if the `autoQueue` option is set to false.

## Signature

```javascript
/**
 * @returns {Kuzzle} this
 */
startQueuing();
```

## Return

The `Kuzzle` instance.
