# stopQueuing
Stops the requests queuing. Works only in offline mode, and if the [autoQueue]({{ site_base_path }}sdk-reference/js/5/kuzzle/#properties) option is set to `false`.

---

## Return Value

Returns the `Kuzzle` SDK object to allow chaining.
